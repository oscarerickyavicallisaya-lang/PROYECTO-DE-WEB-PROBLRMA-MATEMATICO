01 — Contexto

El problema que resolvemos

🌿

Fibonacci en la naturaleza y las finanzas

La serie de Fibonacci aparece en el crecimiento de poblaciones, la reproduccion de especies y en estrategias de ahorro progresivo. Cada numero es la suma de los dos anteriores: 0, 1, 1, 2, 3, 5, 8, 13...

🔐

Primos en la seguridad informatica

Los numeros primos son la base de la criptografia moderna. Un numero primo solo es divisible entre 1 y si mismo. Son ideales para generar claves seguras y validar codigos de acceso.

⚡

Que hacemos aqui?

Generamos la secuencia de Fibonacci y detectamos cuales de esos valores tambien son numeros primos. Combinamos ambos conceptos para revelar los "Numeros de Fibonacci Primos".

02 — Algoritmo

Como funciona?

1

Generar Fibonacci

Iniciamos con a = 0 y b = 1. En cada iteracion calculamos c = a + b, luego avanzamos: a = b, b = c. Sin arrays.

2

Verificar si es primo

Contamos cuantos divisores tiene el numero. Si el contador es exactamente 2 (solo 1 y el mismo), es primo.

3

Mostrar resultados

Mostramos la secuencia completa y resaltamos en naranja los terminos que tambien son numeros primos.

JavaScript — algoritmo base
let a = 0, b = 1, c;

for (let i = 0; i < n; i++) {
  // Contar divisores
  let contador = 0;
  for (let j = 1; j <= a; j++) {
    if (a % j == 0) contador++;
  }
  let primo = (contador == 2);

  // Avanzar sin arrays
  c = a + b; a = b; b = c;
}

04 — Conclusion

Por que importa esto?

La serie de Fibonacci y los numeros primos no son simples curiosidades matematicas: son herramientas que la naturaleza y la tecnologia han adoptado por razones muy concretas.

Un plan de ahorro basado en Fibonacci es psicologicamente sostenible porque los incrementos son graduales y siguen un patron natural. Al mismo tiempo, identificar cuales de esos montos son primos nos da valores con propiedades criptograficas utiles.

Las matematicas estan en nuestros ahorros, en la seguridad de nuestras contrasenas y en los patrones que vemos en la naturaleza.
