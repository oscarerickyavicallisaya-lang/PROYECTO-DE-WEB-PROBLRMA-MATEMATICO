01 — Contexto

El problema que resolvemos

🌿

Fibonacci en la naturaleza y las finanzas

La serie de Fibonacci aparece en el crecimiento de poblaciones, la reproducción de especies, los patrones de flores y hojas, y también en estrategias de ahorro progresivo. Cada número es la suma de los dos anteriores: 0, 1, 1, 2, 3, 5, 8, 13…

🔐

Primos en la seguridad informática

Los números primos son la base de la criptografía moderna. Un número primo solo es divisible entre 1 y sí mismo. Esto los hace ideales para generar claves seguras y validar códigos de acceso.

⚡

¿Qué hacemos aquí?

Generamos la secuencia de Fibonacci y detectamos cuáles de esos valores también son números primos. Esto combina ambos conceptos y revela los llamados "Números de Fibonacci Primos", una rareza matemática fascinante.

02 — Algoritmo

¿Cómo funciona?

1

Generar Fibonacci

Iniciamos con a = 0 y b = 1. En cada iteración calculamos c = a + b, luego avanzamos: a = b, b = c.

2

Verificar si es primo

Contamos cuántos divisores tiene el número. Si el contador es exactamente 2 (solo 1 y él mismo), ¡es primo!

3

Combinar resultados

Mostramos toda la secuencia Fibonacci y resaltamos con color cuáles de esos términos también son números primos.

JavaScript simplificado
let a = 0, b = 1, c;

for (let i = 0; i < n; i++) {
  // ¿Es primo?
  let contador = 0;
  for (let j = 1; j <= a; j++) {
    if (a % j == 0) contador++;
  }
  let esPrimo = (contador == 2);

  // Avanzar secuencia
  c = a + b;
  a = b;
  b = c;
}

04 — Conclusión

¿Por qué importa esto?

La serie de Fibonacci y los números primos no son simples curiosidades matemáticas: son herramientas que la naturaleza y la tecnología han adoptado por razones muy concretas.

Un plan de ahorro basado en Fibonacci es psicológicamente más sostenible porque los incrementos son graduales y siguen un patrón natural. Al mismo tiempo, identificar cuáles de esos montos son primos nos da un subconjunto de valores con propiedades criptográficas útiles.

Este proyecto demuestra que las matemáticas no están separadas de la vida cotidiana: están en nuestros ahorros, en la seguridad de nuestras contraseñas y en los patrones que vemos en la naturaleza.
